# Farma Scale — HTML pronto para GitHub

Estrutura correta:

- index.html
- assets/
  - imagens dos prints dos anúncios

Importante:
1. Suba a pasta `assets` junto com o `index.html`.
2. No GitHub, confira se os arquivos ficaram exatamente em:
   `assets/ad-creative-...png`
3. Se usar GitHub Pages, publique a partir da raiz do repositório ou da pasta que contém `index.html` e `assets`.

Não renomeie nem mova a pasta `assets` sem atualizar os caminhos no HTML.
